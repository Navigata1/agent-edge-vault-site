Build Standards

Motion with a budget

Everything animated on this page obeys four rules — so the art compounds credibility instead of costing it.

RULE 01

Reduced motion, respected

Every animation on this page collapses gracefully when the visitor asks for stillness. Try it — the content survives intact.

RULE 02

Canvas sleeps offscreen

The sea and the gem only render while visible. No hidden loops burning frames behind the scroll.

RULE 03

Content before chrome

Headline, copy, and links are plain HTML — readable by crawlers, screen readers, and no-JS contexts. No more counters stuck at zero.

RULE 04

One signature per page

Each surface earns one memorable moment. Everything around it stays quiet, so the moment lands.
